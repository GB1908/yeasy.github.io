A simple "access analyzer"

This page explains a simple application of gnuplot for WWW access. The HTTP Server adopted here is Apache. An access-log file of the Web server is processed with a Perl program, then it is depicted. This method can be used for an automatic generation of graph image whose data are stored in a text file.
