WWW Access Logの簡単なグラフ化

ここでは，gnuplotをWWWで使う簡単な方法として，WWW Serverのlog file を集計して図にする方法を取り上げます．HTTP ServerとしてApacheを想定しています．このlogファ イルをperlで処理した後，gnuplotを使ってサーバへのアクセスの統計を図示 してみましょう．ここでの方法は，テキストファイルに保存された数値データ をperlで処理し，自動的にグラフの画像を作成するのに応用できます．
